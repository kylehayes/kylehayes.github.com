HTML5 Forms

Apr 13th, 2010 | Comments

With all the buzz of HTML5, most of it has been centered around the new video and canvas tags, however, so much more is being added and spec’ed out. One particular area that has been given some much needed love is forms.

Some new form input types that we get are:

  • search
  • tel
  • url
  • email
  • datetime, date, month, week, time, and datetime-local
  • number
  • range
  • color

Along with that, some validation. But what browsers are supporting this new functionality? Check out the litmus tests here: http://cl.ly/4Bq

Update: See if your browser passes the HTML5 “Acid” test: http://html5test.com/
